# Setup

- Create virtual environment
- Activate virtual environment
- Install dependencies

On Windows

```
python3 -m venv .venv
./.venv/bin/activate
python3 -m pip install -r requirements.txt
```

On Linux

```
python3 -m venv .venv
source ./.venv/bin/activate
python3 -m pip install -r requirements.txt
```
